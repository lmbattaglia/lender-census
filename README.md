# lender-census

This Github repository provides a general script to extract data from the American Community Survey for the purposes of the Lender Center for Social Justice's Closing the Racial Wealth Gap dashboard project. Included in the repository are the general fetcher script, and two more specific example scripts with their corresponding Excel sheets. 

To use the general fetcher script, a user needs 3 inputs:

1. API Key (can sign up for a key on the Census website)
2. The specific name of the table they would like to pull data from (eg. S2501)
3. The latest year that data is available for (eg. 2024)

*Claude was used in the creation of this script and troubleshooting of errors.*
